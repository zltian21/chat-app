## Structure

#### Tech Stack

- React.js
- Typescript
- React Redux
- React Router

#### Pages

- `/login`: `<Login>`
- `/register`: `<Register>`
- `/main`: `<Main>`, consisted of `<ChatPanel>`, `<RoomDrawer>`, `<RoomList>`, `<Sidebar>`, `<NotificationList>`.

#### Components

- `<Layout>`: Layout for the whole app.
- `<Message>`: Each chat message.
- `<Room>`: Each room info displayed in the rooms list.
